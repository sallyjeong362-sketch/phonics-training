# 소리쑥쑥 파닉스 (phonics-training)

처음 파닉스를 시작하는 초등학생부터, 아직 단어 읽기가 어려운 중·고등학생까지 쓸 수 있는 웹 기반 파닉스 학습 앱입니다. 설치 없이 브라우저에서 바로 실행되는 단일 `index.html` 파일로 되어 있습니다.

## 기능

1. **✏️ 알파벳 따라쓰기** — 대문자/소문자 A~Z를 손가락(터치)이나 마우스로 따라 쓰면 자동으로 채점해서 별점(⭐~⭐⭐⭐)을 줍니다. 글자 이름과 글자 소리(파닉스 발음)를 들을 수 있고, **"▶️ 획순 보기"** 버튼을 누르면 각 획을 번호와 화살표로 순서대로 애니메이션으로 보여줘 올바른 필순을 익힐 수 있습니다 (대문자·소문자 52자 모두 지원).
2. **🔍 대문자·소문자 테스트** — 따라쓰기를 연습한 뒤, 대문자를 보고 짝꿍 소문자를 고르는(또는 반대) 10문제 테스트. b/d/p/q처럼 헷갈리는 글자가 보기로 나옵니다.
3. **🃏 짝꿍 카드 게임** — 테스트를 게임화한 메모리 카드 게임. 카드를 뒤집어 대문자–소문자 짝을 맞추고, 적은 횟수로 완료할수록 별을 더 받습니다.
4. **🔊 소리 합치기 (블렌딩)** — `cat`이면 `c·a·t` 각 철자 밑에 점(●)이 있고, 점을 누르면 "크·애·트" 소리가 하나씩 재생됩니다. 점들을 왼쪽에서 오른쪽으로 쭉 드래그하면 소리가 이어지며 마지막에 "캣" 전체 발음이 재생됩니다.
   - 단계: 단모음 a/e/i/o/u → 짝소리(sh·ch·th·ng) → **자음 블렌드(bl·cr·st·tr 등)** → 매직 e → 모음 친구(ai·ee·oa·oo) → **r 소리 친구(ar·or·er)**, 총 10단계·116개 단어
   - 자음/모음/이중글자를 색깔로 구분하고, 한글 발음 힌트를 켜고 끌 수 있습니다.
5. **🧩 청크 읽기 (중·고등용)** — `in·for·ma·tion`처럼 긴 단어를 덩어리(chunk) 단위로 색깔로 나눠 시각화합니다. 덩어리를 하나씩 누르고 드래그로 이어 읽으며, `-tion`, `-ture`, `-ful`, `-ous` 같은 자주 나오는 끝소리 패턴도 배웁니다. 총 4단계·50개 단어이며, 마지막 단계에는 `responsibility`(6조각)처럼 어려운 단어도 포함됩니다.

## 사용 방법

- `index.html`을 브라우저로 열면 바로 실행됩니다 (Chrome/Edge/Safari 권장).
- 발음은 기기에 내장된 영어 음성(Web Speech API)으로 재생됩니다. 소리가 나지 않으면 무음 모드를 해제하세요.
- 학습 기록(별점, 최고 점수, 완료한 단어)은 브라우저의 `localStorage`에 저장됩니다.

## 배포

`main` 브랜치에 push되면 GitHub Actions(`.github/workflows/deploy-pages.yml`)가 GitHub Pages로 자동 배포합니다.
